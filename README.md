## jsdelivr

```
加载任何GitHub版本，提交或分支
https://cdn.jsdelivr.net/gh/besscroft/cdn@version/file

获取最新版本
https://cdn.jsdelivr.net/gh/besscroft/cdn/file

在末尾添加/以获取目录列表
https://cdn.jsdelivr.net/gh/besscroft/cdn/

//将“.min”添加到任何JS/CSS文件以获取缩小版本
//如果不存在，会自动生成
https://cdn.jsdelivr.net/gh/besscroft/cdn@version/.min.file
```
